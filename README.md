Этот проект реализует систему управления грузоперевозками с графическим интерфейсом Windows Forms. Работа демонстрирует переход от консольных приложений к полноценным десктопным программам.
Приложение позволяет управлять клиентами и тарифами транспортной компании: добавлять, редактировать и удалять записи. Данные можно загружать из CSV-файлов и сохранять в них. Таблицы поддерживают сортировку по столбцам.
Проект состоит из двух частей. Первая включает упражнения на C++: изучение множественного наследования, перегрузки операторов и работы с объектами. Вторая часть — готовое приложение с интуитивным интерфейсом, валидацией данных и модульной архитектурой.
Работа выполнена в Санкт-Петербургском политехническом университете. Проект показывает практическое применение принципов объектно-ориентированного программирования при создании пользовательских приложений.
